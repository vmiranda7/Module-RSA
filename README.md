
#Module RSA

We can do all of this basinc things with RSA:
- Encrypt
- Decrypt
- Sign
- Verfy
- Blind


####Package used:
~ Bignum [Bignum npm page] (https://www.npmjs.com/package/bignum).

``
npm-install-all
``
####Basic commands are:
``
node index.js
